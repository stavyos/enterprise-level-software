# PR-10: Orchestrate Jenkins with Docker Compose

## Purpose
This Pull Request integrates Jenkins into the project's core `docker-compose.yaml` file. This allows developers to start the entire CI/CD and database stack with a single command, ensuring networking and persistence are handled automatically.

## Reviewer Reading Guide
1. **`docker-compose.yaml`**: Review the new `jenkins` service definition and the shared `enterprise-network`.
2. **`docs/infrastructure/jenkins.md`**: Check the updated startup instructions.

## Key Changes
- **Docker Compose Integration**: Added `jenkins` service with Docker socket mounting and persistent volume.
- **Unified Networking**: Explicitly defined `enterprise-network` to allow seamless communication between Jenkins and TimescaleDB instances.
- **Documentation Update**: Simplified the Jenkins setup guide to use the Docker Compose workflow.
